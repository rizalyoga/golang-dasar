# Langkah - langkah project:

1. Buat database.
2. Install package yang dibutuhkan :

- install gin :
  - `go get github.com/gin-gonic/gin `
- install pgx driver:
  - ` go get github.com/jackc/pgx/v5/stdlib`
- install UUID:
  - `go get github.com/google/uuid`
- install go crypto:
  - `go get golang.org/x/crypto`
